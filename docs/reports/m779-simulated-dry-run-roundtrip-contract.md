# M779 Simulated Dry-Run Roundtrip Contract

M779 defines the full simulated dry-run roundtrip for NODAL OS.

The contract is `SIMULATED_DRY_RUN` only. It binds negative and positive simulated flows through request intake, fixture loading, policy gate, optional manual approval, decision, evidence envelope, redaction proof, ledger projection, no-execution proof, and response return.

No real runtime execution, provider/cloud live call, filesystem write, browser automation, capability unlock, public release, Chrome Web Store submission, signed public ZIP, product file change, Bridge change, or CSP change is allowed.
