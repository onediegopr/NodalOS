# M667 Store Disclosure Prep

Decision: `M667 CERRADO / STORE_DISCLOSURE_PREP_READY`

## Scope

M667 prepares Chrome Web Store disclosure material. It does not publish, upload, sign, create a final package, or mark public release ready.

## Permission Disclosure Draft

The public candidate uses local host permissions for `127.0.0.1` and `localhost`, plus extension permissions for side panel, storage, alarms, tabs, scripting, and activeTab. Final disclosure must explain why each permission exists and what is not enabled.

## Privacy Disclosure Draft

NODAL OS public candidate is local-first. Runtime/productive execution, provider/cloud, filesystem/workspace, browser automation, and capability unlock remain disabled. Evidence must not include secrets, API keys, raw long logs, or user data.

## Support Disclosure Draft

Support URL and privacy URL are placeholders until final listing prep. Internal support process and rollback plan must be validated before public release.

## Listing Gaps

Open before Store readiness:

- Final listing name.
- Final screenshots/assets.
- Final support URL.
- Final privacy URL.
- Manual QA evidence.
- Public package candidate evidence.
- Permission warning screenshots.
