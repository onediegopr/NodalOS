# ADR: Licensing/Governance Foundation M37

## Status

Accepted.

## Context

M37 defines plans, licenses, limits, feature flags, and governance checks before productization. It is contract-first and does not implement payment collection, real SaaS activation, real license emails, or external user provisioning.

## Decision

Add licensing/governance contracts:

- `NexaPlan`
- `NexaPlanKind`
- `NexaLicense`
- `NexaLicenseStatus`
- `NexaLicenseEntitlement`
- `NexaFeatureFlag`
- `NexaUsageLimit`
- `NexaUsageCounter`
- `NexaLicenseDecision`
- `NexaLicenseAuditEvent`
- `NexaLicensePolicyEvaluator`

## Plans

Plans are:

- Free
- Trial
- Pro
- Enterprise

Free:

- generated from a controlled request;
- requires email;
- one free license per email window;
- maximum default duration: 7 days;
- strong limits;
- does not enable sensitive real pilots;
- does not enable productive vault;
- does not enable raw profile.

Trial:

- generated by admin/controlled flow;
- configurable features and limits;
- configurable duration;
- can be restricted by account/workers;
- can be suspended/revoked by status.

Pro:

- higher limits;
- external low-risk features may be included;
- sensitive sites remain disabled by default.

Enterprise:

- multi-worker;
- organization-level feature flags;
- governance support;
- extended audit;
- sensitive sites still require explicit compliance policy.

## Feature Flags

Initial features:

- BrowserRuntime
- CdpLiveReadOnly
- SafeDownload
- SafeUpload
- RecorderReadOnly
- ReplaySafeMode
- DocumentWorkflowSandbox
- SensitiveSimulation
- ExternalLowRiskAuth
- SensitiveRealPilot
- ProductiveVault
- ProfileControlled
- AdminConsole
- RecorderProductive
- ReplayProductive

Disabled by default:

- SensitiveRealPilot
- ProductiveVault
- RecorderProductive
- ReplayProductive

`ProfileRaw` is not exposed as a feature.

## License Policy

The evaluator checks:

- account validity;
- account status;
- license status;
- expiration;
- feature inclusion;
- worker authorization;
- usage limits;
- enterprise/compliance requirement for sensitive features.

It fails closed for:

- expired license;
- disabled feature;
- exceeded limit;
- unauthorized worker;
- suspended/expired account;
- sensitive feature without enterprise/compliance policy.

## Audit

License decisions emit redacted audit events without secrets, cookies, sessions, payment card data, or sensitive document contents.

## What Remains Blocked

M37 does not implement:

- real billing;
- real payment gateway;
- real SaaS activation;
- real license emails;
- productive vault;
- sensitive real pilot;
- productive recorder/replay;
- raw profile.

## Future Work

Future milestones can add persistence, admin UI, billing integration, renewal workflows, tenant provisioning, and support workflows after security/compliance review.
