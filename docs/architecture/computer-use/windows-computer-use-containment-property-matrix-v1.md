# Windows Computer Use Containment Property Matrix v1

## Scope

This matrix is a negative-property lock for WCU containment. It does not reopen `WCU-031-036`, does not authorize `WCU-037-044`, and does not implement live read-only collection.

Source of truth in code: `ComputerUseContainmentPropertyCatalog`.

## Enforcement Rule

Containment PASS is not live GO. Locator confidence, OCR, UIA events, Win32 context, evidence, and design gates cannot grant action authority.
