# Recipe Template System-specific Detail Summaries

Product-surface phase: 2/4.

System summaries are read-only product explanations for Global and LATAM template packs.

## Excel / Microsoft 365

Explains preview/fixture-safe data workflows. No real Microsoft 365 connector, file mutation, or live sync is enabled. Evidence and validation are expected by reference.

## Google Workspace

Explains Google Sheets, Drive, Gmail, and Calendar templates as draft/review workflows. No Google API call or Gmail delivery is enabled.

## SAP

Explains SAP templates as future API/connector-first. SAP GUI automation, RFC, BAPI, and OData calls are not enabled. Mutation-like drafts remain human-gated and live-blocked.

## Mercado Libre / Mercado Pago

Explains marketplace/payment templates as review-only. No API call, payment execution, stock update, price update, listing publication, message delivery, or claims mutation is enabled. Human review is required.

## ARCA / Fiscal Argentina

Explains fiscal templates as legal/fiscal review-only. No fiscal submission, ARCA/AFIP web service call, certificate use, or private-key use is enabled. Human/legal review is required.

## ERP Local LATAM

Explains ERP local templates as draft/review only. No real ERP connector, API call, desktop automation, or mutation is enabled. Metadata may mention Tango, Bejerman, Contabilium, Alegra, Siigo, Odoo, TOTVS, CONTPAQi, and Aspel.

## Generic Browser Portals

Explains browser portal templates as readiness checks, previews, or playbooks. No browser automation, real login, or challenge bypass is enabled.

## Computer Use Legacy

Explains legacy desktop templates as manual playbooks or drafts. No desktop automation, UIA/vision execution, hotkey hook, or OS listener is enabled.
